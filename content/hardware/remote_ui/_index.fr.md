## Qu'est-ce que Calaos Remote UI ?

Calaos Remote UI est un système permettant de connecter des **écrans tactiles embarqués** directement à votre serveur domotique Calaos. Ces petits panneaux muraux deviennent des interfaces de contrôle distribuées dans votre installation : vous pouvez y visualiser l'état de vos équipements et y envoyer des commandes en temps réel, sans passer par une application mobile ou un navigateur web.

Les écrans se provisionnent automatiquement sur le serveur Calaos via Wi-Fi ou Ethernet et communiquent de façon sécurisée grâce à une authentification HMAC-SHA256.

## Carte supportée

Calaos supporte actuellement la carte **ESP32-P4 Smart 86 Box** de Waveshare — un panneau 86 mm au format standard avec un écran IPS tactile 4 pouces (720 × 720 pixels), basé sur le microcontrôleur ESP32-P4 haute performance avec Wi-Fi et Ethernet.
